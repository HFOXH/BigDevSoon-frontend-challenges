---

## 🎯 Objetivo del repositorio

Este repositorio forma parte de mi práctica continua de **Frontend Development** mediante desafíos de UI.

El objetivo es mejorar habilidades en:

- HTML semántico
- CSS moderno
- Layouts responsive
- Componentes UI reutilizables
- Interactividad con JavaScript

Además, está abierto para la comunidad. Puedes explorar, reutilizar o adaptar el código en tus propios proyectos.
